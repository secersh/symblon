# Bug Squasher

> Hunt down bugs and close them before they multiply.

Two symbols in this pack:

| Symbol | Criteria |
|---|---|
| First Blood | Close your first bug-labelled issue |
| Bug Squasher | Close 5 bug-labelled issues within 48 hours |

Available in **scouts**, **wizards**, and **hackers** themes.
